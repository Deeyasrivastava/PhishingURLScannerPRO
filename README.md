PhishingURLScannerPRO

A simple and effective **Python-based phishing URL detection tool** that analyzes URLs using rule-based heuristics.  
This project helps detect potentially malicious or suspicious URLs using multiple checks such as domain patterns, unusual ports, subdomains, punycode, suspicious keywords, URL shorteners, and more.



Features
✔ Analyze **single URLs**  
✔ Analyze URLs from **a text file (urls.txt)**  
✔ Detect **suspicious keywords**  
✔ Detect **raw IP-based links**  
✔ Detect **URL shorteners** (bit.ly, tinyurl, t.co, etc.)  
✔ Identify **punycode (IDN)** domains  
✔ Identify **deep/multiple subdomains**  
✔ Detect **HTTP vs HTTPS**  
✔ Flag **weird ports** (non-standard ports)  
✔ Flag **long/encoded URLs**  
✔ Export results to **results.csv**  
✔ Simple **terminal menu UI**  
✔ Runs completely **offline**
